# Java_Chip8_Emu
 a work in progress chip 8 emulator in java withou using any external libaries
 
# Controls

chip8              
| 1 | 2 | 3 | C |   
| 4 | 5 | 6 | D |   
| 7 | 8 | 9 | E |   
| A | B | 0 | F |   

keyBoard

| 1 | 2 | 3 | 4 |
| Q | W | E | R |
| A | S | D | F |
| Z | X | C | V |
